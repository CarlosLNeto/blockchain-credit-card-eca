# ECA Frontend

Interface web para portadores de cartão ECA.

## Instalação

```bash
npm install
```

## Execução

```bash
npm start
```

Interface: http://localhost:3000

## Estrutura

- `src/components/` - Componentes React
- `src/pages/` - Páginas da aplicação
- `src/services/` - Integração com API
- `src/context/` - Gerenciamento de estado

## Tecnologias

- React 18
- Material-UI
- React Router
- Axios
