# ECA Backend

API REST para a bandeira de cartão ECA baseada em blockchain.

## Instalação

```bash
npm install
```

## Execução

```bash
npm start
```

Servidor: http://localhost:3001

## Estrutura

- `src/blockchain/` - Implementação da blockchain
- `src/models/` - Modelos de dados
- `src/controllers/` - Lógica de negócio
- `src/routes/` - Rotas da API
- `src/middleware/` - Autenticação JWT
- `src/config/` - Configurações

## Tecnologias

- Node.js + Express
- Blockchain customizado
- JWT + Bcrypt
- SHA-256
